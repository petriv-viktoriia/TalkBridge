# TalkBridge

## Мета проєкту

Розробити веб-платформу, яка дозволяє користувачам знаходити носіїв іноземних мов для практики спілкування, обирати відповідного партнера за заданими критеріями (мова, рівень володіння, рейтинг тощо) та здійснювати спілкування в режимі реального часу.

---

## Основна ідея

Платформа поєднує функціонал соціальної мережі та сервісу пошуку мовних партнерів.  
Користувач може:

- створювати профіль із зазначенням мов, рівня володіння, інтересів;
- переглядати профілі інших користувачів;
- фільтрувати пошук за мовою, рівнем, рейтингом, країною;
- переглядати рейтинг і відгуки про інших користувачів;
- спілкуватися через **чат у реальному часі** (реалізація через WebSocket).

---

## Основні функціональні можливості

### 1. Реєстрація та автентифікація користувачів
- створення облікового запису
- можливість редагувати профіль (фото, мови, опис, рівень володіння та інше).

### 2. Стрічка
- перегляд стрічки з користувачами;
- фільтрація та пошук за мовами, рівнем, рейтингом, країною тощо;
- відображення короткої інформації про користувача, рейтингу, відгуків.

### 3. Система рейтингів та відгуків
- після спілкування користувач може залишити оцінку та короткий відгук;
- формування середнього рейтингу користувача.

### 4. Спілкування в чаті
- обмін повідомленнями через **WebSocket**;
- статус “онлайн/офлайн”;
- історія чатів зберігається в базі даних.

### 5. Адміністративна панель (опціонально)
- керування користувачами;
- модерація профілів та відгуків.

---

## Технологічна частина (приклад стеку)

- **Back-end:** NestJS
- **Front-end:** React / Angular  
- **База даних:** PostgreSQL / MySql
- **Аутентифікація:** OAuth2  
- **Додатково:** WebSocket, Docker
